# The Devil's Hour

A 2D top-down puzzle-solving role-playing horror game developed with SDL 2 library.

[Demo](https://youtu.be/YZsIefdKz60)

### Prerequisites
- GCC
- CMAKE (www.cmake.org)
- SDL 2.0 (www.libsdl.org)
- SDL2\_image
- SDL2\_gfx
- SDL2\_ttf
- SDL2\_mixer

### Run the game
```
mkdir Debug
cd Debug
cmake ..
make
./DevilsHour
```
